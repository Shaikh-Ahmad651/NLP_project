# NLP Project – Slang Detection & Ambiguity Analysis

## 📌 Overview
This project is a Natural Language Processing (NLP) application that analyzes input sentences to:
- Detect slang words
- Identify word-level ambiguity

It is designed to help in understanding informal language and ambiguous meanings in text.

---

## 🚀 Features
- ✅ Slang detection in sentences
- ✅ Word-level ambiguity identification
- ✅ Simple command-line interface
- ✅ Easy to extend with more NLP features

---

## 🛠️ Technologies Used
- Python
- Basic NLP techniques (tokenization, dictionary lookup, etc.)
